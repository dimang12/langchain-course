What's in the RAG Pipeline document?
The RAG_Pipeline_Flow.md document describes a Retrieval-Augmented Generation pipeline with these stages:

1. Document ingestion and chunking
2. Embedding generation via OpenAI
3. Vector storage in ChromaDB
4. Semantic search and retrieval
5. LLM response generation with context
Can you summarize the auth flow?
The JWT_Auth_Flow.md covers a token-based authentication system: users authenticate via login endpoint, receive a JWT access token (15min) and refresh token (7d), and include the bearer token in subsequent API requests. Token refresh happens automatically before expiry.
